To run this file, follow these steps:
1. Have the both of the datasets in their own folder that is placed in the same directory of the PageRank.py file

ie:
  ->DATASETS
      -CA-GrQc.txt
      -com-dblp.ungraph.txt
  -PageRank.py

2. To change the dataset being used in the code, change the filePath variable located in the data frame (df) variable from "filePath" to "bonusFilePath"
